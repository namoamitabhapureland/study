## 核心功能

### 化生單字修煉
透過遊戲化方式自然記憶單字，適合長期堅持。

### 高中全真模考
完整歷屆高中英文衍生模擬考試，幫助穩固基礎。

### 托福適性考場
動態聽說讀寫練習，模擬真實考試環境。

### 多益職場實戰
200題職場英文實戰，提升職場競爭力。
